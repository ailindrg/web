Esta pequeña empresa inicio como proyecto escolar a
principios del 2025, con el fin de aprovechar lo antes
aprendido para ponerlo en practica con este prototipo
de pagina web, de venta de productos para el cabello.

En esta pagina encontraras diferentes apatados, donde
podras encontrar la variedad de opciones que esta pagina
tiene para ti, desde shampoos y cremas para peinar, hasta
geles y aceites para tu cabello, descripciones de cada
producto hasta promociones que no puedes dejar pasar.
